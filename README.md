<h1>Cinema Hall</h1>
<b>Uses TMDB top rated movies API to fetch movies. User can see movie details and add movies to wishlist</b>br>

<h3>Technologies Used:</h3>
<b>Retrofit2, OKhttp3 | Room Database | Glide | Paging 3 </b>
